# Reelax — Assignment

React JS implementation of the Reelax billing/checkout UI from Figma.

## Setup & Run

1. Make sure Node.js (v16+) is installed
2. In the project root, run:

```bash
npm install
npm start
```

The app will open at http://localhost:3000

## Build for Production

```bash
npm run build
```
